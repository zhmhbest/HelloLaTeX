安装

以管理员身份运行install-tl-windows.bat。
安装完成后，确保${INSTALL_LOCATION}\texlive\${VERSION_YEAR}\bin\win32在环境变量中。

仓库

tlshell
REM 选项 -> Repositories

更新

REM 查看可用更新
tlmgr update --list

REM 更新全部
tlmgr update --self --all

操作

  • 新建*.tex
  • Ctrl + Alt + B: 编译为PDF
  • Ctrl + Alt + V: 预览PDF
  • Ctrl + Alt + J: 代码对于PDF位置
  • Ctrl + Click on PDF: 跳转到PDF内容对应代码位置
